# 数组

## 将所有值赋值为-1

由于-1 的八位二进制补码是`11111111`，因此可以使用`memset`

```C++
memset(arr, -1, sizeof(memset));
```

# 数学

## 取整

C++中向上取整和向下取整可以调用库函数`ceil()`和`floor()`，但是在处理整数运算时会有较大的性能开销

如何计算 $\left\lfloor{\dfrac{a}{b}}\right\rfloor$ 和 $\left\lceil\dfrac{a}{b}\right\rceil$

- C++中的整数除法本身就是向下取整，因此
  $\left\lfloor{\dfrac{a}{b}}\right\rfloor = a/b$

- 数学证明 $\left\lceil\dfrac{a}{b}\right\rceil=\left\lfloor{\dfrac{a+b-1}{b}}\right\rfloor$，故 $\left\lceil\dfrac{a}{b}\right\rceil=(a+b-1)/b$
